# Employee CRUD Assignment

## 📌 About Project

Java JDBC-based Employee Management System with CRUD operations.

## ⚙️ Features

* Add Employee
* View Employees
* Update Employee
* Delete Employee

## 🛠️ Tech Stack

* Java
* JDBC
* PostgreSQL
* Eclipse

## 🚀 How to Run

1. Clone repo
2. Open in Eclipse
3. Add PostgreSQL JDBC jar
4. Update DB config
5. Run main file
